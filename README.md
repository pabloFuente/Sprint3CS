# Sprint3CS

Need:
  - Ruby 2.2.2 or higher
  - npm 4.1.2 
Install:

```bash
  $ gem install cucumber
  $ gem install bundler
  $ npm install phantomjs
```

Run test:
 - Selenium
  ```bash
    $ cd selenium/
    $ bundle install
    $ cucumber 
  ```
 - Poltergeist
  ```bash
    $ cd poltergeist/
    $ bundle install
    $ cucumber 
  ```
